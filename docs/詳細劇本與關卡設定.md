# 🍕 義次元腦洞大開：校園瑪瑪咪呀合成大戰 🤯 - 詳細劇本與關卡設定

為了讓國小學童能有最佳的遊戲體驗，本劇本將原版角色故事中過於黑暗、暴力的元素（如戰爭、不良字眼）進行了**「全年齡向（Kid-friendly）與迷因化（Meme/Brainrot）」**的改編，保留了角色荒謬、搞笑的外觀與核心特色，並融入時下流行的網路熱梗（如：Sigma、破防、超派等），打造出充滿歡樂與吐槽點的尋寶遊戲！

## 一、 角色背景設定與人物關係（校園迷因版）

### 1. 🥷 Cappuccino Assassino (卡布奇諾忍者 - Capu)
* **外觀：** 長著眼睛與腿的卡布奇諾咖啡杯，手持「會發熱的武士刀」，頭戴火影忍者護額。
* **背景：** 瑪瑪咪呀學園的暗影守護者，專門懲罰「不喝咖啡的人」和「偷吃營養午餐的人」。他是個充滿「中二病」的忍者，認為自己是校園的 Sigma (獨行俠)。
* **關係：** **Ballerina Cappuccina** 的老公。最近因為沉迷修煉忍術冷落了老婆，老婆正在生氣。常被太妹 **Espressona Signora**（濃縮咖啡大姐頭）搭訕但總是瘋狂閃躲。

### 2. 🩰 Ballerina Cappuccina (卡布奇諾芭蕾女伶)
* **外觀：** 穿著芭蕾舞裙的優雅咖啡杯。
* **背景：** Capu 的老婆，學園的舞蹈社社長，口頭禪是「mi mi mi mi！」。超級熱愛跳舞，只要聽到音樂就會忍不住轉圈圈，但最近她的專屬舞鞋被藏起來了，導致她無法跳舞而大發脾氣（大破防）。
* **關係：** 對老公 Capu 只顧著練劍很不滿；視 Espressona Signora 為死對頭。聽說她最近很粉一個叫「Ballerino Lololo」的神秘蒙面舞者。

### 3. 🐘 Lirili Larila (仙人掌大象 - 哩哩哩·啦哩啦)
* **外觀：** 擁有大象頭、仙人掌身體，腳上穿著巨大海灘拖鞋的奇妙生物。身邊飄著一個神秘的鬧鐘。
* **背景：** 從沙漠轉學來的交換學生，極度渴望「水」跟「免費的 Wi-Fi」。擁有用鬧鐘「控制時間（暫停或快轉）」的超能力，但他最常用這個能力來多睡五分鐘。
* **關係：** 常被搭乘藍色熱氣球的「熱氣球阿北 (Uncle Ramon)」吐槽為什麼仙人掌要穿拖鞋。跟 Tung Tung 是好朋友。

### 4. 🪵 Tung Tung Tung Sahur (邦邦棒球木頭人 - 咚咚咚)
* **外觀：** 一根長著手腳的木頭，手持巨大的棒球棒。
* **背景：** 負責早上叫大家起床的「校園鐘聲管理員」。如果他敲了三次「咚咚咚」你還賴床，他就會變身成巨大的「木頭大猩猩」衝到你面前大吼。是個力氣超大但內心溫柔的傻大個。
* **關係：** Lirili 的好兄弟。因為力氣太大，上次不小心把學校的「傳說級披薩食譜」給打飛散落各處。

### 5. 🐊 Bombardilo Crocodilo (水狼牙鱷魚飛機 - 轟炸鱷鱷)
* **外觀：** 一顆超大鱷魚頭長在轟炸機上。
* **背景：** 學校的「惡作劇大王」。原本的愛好是丟炸彈，現在被校長勒令只能丟「超派水球」跟「史萊姆黏液」。性格調皮搗蛋，會在天上飛來飛去空投水球捉弄學生。
* **關係：** 常在上空看見底下的鬧劇，掌握了許多別人不知道的秘密。最近覺得 Tralalero 買的新鞋很帥，想用水球砸他。

### 6. 🐒 Brr Brr Patapim (巨足長鼻猴樹人 - 帕塔平)
* **外觀：** 長鼻猴與樹木的結合體，腳超級無敵巨大，頭上長著小手。
* **背景：** 校園後山生態園區的管理員。非常善良但運氣很差。他得到了一頂酷炫的金色帽子，裡面卻住著一隻會一直發出「Brrr Brrr」怪聲的藍色青蛙 Slim，讓他每天都精神衰弱。
* **關係：** 如果有危險，他會用巨大的腳把壞人踩扁。他正在尋找魔法師 Tiramisu（提拉米蘇大師）幫他把青蛙弄走。

### 7. 🦈 Tralalero Tralala (潮鞋拳擊鯊 - 啦啦鯊)
* **外觀：** 一隻站立的藍色鯊魚，側鰭變成腿，腳穿聯名款 Nike 潮鞋，手戴紅色拳擊手套。
* **背景：** 瑪瑪咪呀學園的體育股長兼超級潮男。非常愛惜他的 Nike 鞋（誰踩到他就會瘋狂暴走）。脾氣暴躁但很講義氣。
* **關係：** 他的死對頭是邪惡的「漢堡魔王 (Burger)」。最近魔王用巨型餿水攻擊了他的家，還把他的好朋友 Arturo 抓走了，他正戴上拳擊手套準備去復仇！

## 二、 完整主線劇情：尋找終極瑪格麗特披薩與拯救 Wi-Fi 📡🍕

**【故事背景 (開場影片)】**
「瑪瑪咪呀學園」遇到史上最大危機！邪惡的 **「漢堡魔王 (Burger)」** 入侵了校園！
他不滿大家天天吃義大利麵跟披薩，於是施展了邪惡魔法：
1. 把學校賴以維生的 **「免費 Wi-Fi 分享器」** 藏了起來！(大家都崩潰了，Lirili Larila 更是直接失去生存動力)
2. 把能夠鎮壓魔王的 **「終極瑪格麗特披薩」** 的食譜與材料（麵粉、水、番茄、起司、羅勒葉等）打散，變成交給 7 位被洗腦/遇上麻煩的守護者（也就是我們的 7 位角色）保管。

**【小隊任務與遊戲機制】**
1. **食材收集階段 (60分鐘)**：小朋友們化身為「Pizza 特攻隊」，在 60 分鐘內自由探索校園，前往 7 個角色關卡解謎、完成任務，盡可能收集所有的食材卡牌與 Wi-Fi 密碼碎片。
2. **集合與合成階段**：時間到後全體集合，現場將設置一台「實體合成器」。小隊掃描合成器後，系統會檢查是否集齊 5 種食材（麵粉、水、番茄、起司、羅勒葉）；只要食材齊全即可進行加工，不需先額外合成生麵團。
3. **合作加工階段**：若單一小隊食材不足，可以尋找其他小隊「結盟合作」，共享缺失的食材來完成加工。如果全場都缺少某樣關鍵食材，大會將引導所有小隊「全體大合作」，共同解決謎題拿到食材，推進後續主線並拯救學園！

## 三、 各關卡任務設計 (QR Code 互動內容)

每個角色關卡散佈在校園各處，小隊可以使用手機掃描 QR Code 觸發劇情與任務。

### ⚡ 關卡 1：忍者的修煉 (Cappuccino Assassino)
* **地點建議：** 陰暗的角落或樓梯間。
* **角色對話：** 「Cappu-cappu！吾乃暗影中的卡布奇諾！漢堡魔王用幻術讓我忘記了忍術的結印順序！沒有忍術，我是無法給你披薩材料的，那太不 Sigma 了！」
* **任務內容 (記憶/排列題)：**
  畫面上出現三個忍術手勢（例如：☕咖啡杯、🔥火焰、⚔️武士刀），小朋友必須在校園現場尋找貼在周圍牆上的數字提示，輸入正確的結印順序 (例如 3-1-2)。
* **過關獎勵：** 獲得虛擬寶物卡片 **【陳年特級麵粉】** + 線索：「水球鱷魚知道起司在哪裡。」

### ⚡ 關卡 2：舞鞋在哪裡？ (Ballerina Cappuccina)
* **地點建議：** 穿堂或舞蹈教室外。
* **角色對話：** 「{播放名字音檔} Mi mi mi mi！氣死我了，大破防！有人把我最愛的芭蕾舞鞋藏起來了！沒有舞鞋，我要怎麼跳給 Lololo 看？快幫我找回來！」
* **任務內容 (實境尋寶)：**
  網頁提示：「犯人留下了線索，舞鞋被藏在『有著三棵紅色植物與兩張長椅交界處』。」小朋友必須在附近實際尋找一個貼著「舞鞋」貼紙的小盒子，盒子裡有四位數密碼。將密碼輸入網頁。
* **過關獎勵：** 獲得虛擬寶物卡片 **【神聖番茄】** + 線索：「聽說大腳怪猴子頭上有一隻青蛙。」

### ⚡ 關卡 3：吵鬧的青蛙 (Brr Brr Patapim)
* **地點建議：** 樹下或花園旁。
* **角色對話：** 「{播放名字音檔}... 救命啊...我頭上的金色帽子裡有一隻叫做 Slim 的藍色青蛙一直在叫！提拉米蘇大師說需要一顆『超級充氣球』才能把它氣球般吹走。你們可以幫我嗎？」
* **任務內容 (互動小遊戲)：**
  網頁上出現一顆氣球，小朋友必須要在 10 秒內「狂點擊螢幕 50 下」幫氣球充氣（考驗團隊手速）。充氣成功後，會顯示青蛙「Boom!」被炸飛的動畫。
* **過關獎勵：** 獲得虛擬寶物卡片 **【純淨山泉水】** + 線索：「先收齊五種食材，再回合成器進行合作加工！」

### ⚡ 關卡 4：水球空投警報 (Bombardilo Crocodilo)
* **地點建議：** 操場邊緣或空曠處。
* **角色對話：** 「嘿嘿嘿！我是{播放名字音檔}！今天不丟炸彈，我丟水球！既然你們這些小屁孩覺得自己很聰明，那就來解開我全新強化的無敵邏輯題吧！答錯的話，我就把你們淋成落湯雞，超派！」
* **任務內容 (進階邏輯與推理題)：**
  網頁顯示：「有四顆超級水球：紅、藍、黃、綠，重量都不一樣。已知條件：
  1. 藍色水球不是最重的，也不是最輕的。
  2. 綠色水球比紅色水球重。
  3. 黃色水球比藍色水球輕，但它『不是』最輕的。
  請問從最重到最輕的正確順序是什麼？」
  *(推理過程：黃比藍輕且不為最輕，故藍第二、黃第三。剩下一和四由綠與紅填補，綠大於紅，故綠第一、紅第四)*
  **(答案：綠 > 藍 > 黃 > 紅)**。小隊必須討論出正確順序並輸入，才能成功啟動防護傘閃避空投。
* **過關獎勵：** 獲得虛擬寶物卡片 **【濃郁帕瑪森起司】** + 線索：「大象仙人掌因為沒有 Wi-Fi 在沙漠裡迷路了。」

### ⚡ 關卡 5：沙漠與斷網的絕望 (Lirili Larila)
* **地點建議：** 沙坑或有陽光直射的廣場。
* **角色對話：** 「{播放名字音檔}... 渴死我了，比沒有水更慘的是沒有 Wi-Fi。漢堡魔王在我的時間鬧鐘上鎖了『規律密碼』。你們那麼聰明，能幫我破解數列重啟鬧鐘嗎？」
* **任務內容 (時間進階數列解謎)：**
  網頁顯示五道鎖，前四道鎖上面顯示的時間分別是：
  「第一道：01:15」、「第二道：02:30」、「第三道：04:00」、「第四道：05:45」。
  提示：「漢堡魔王增加的時間間隔越來越長，請問『第五道鎖』必須輸入幾點幾分才能完全解開？」
  *(規律解析：01:15 到 02:30 加了 1小時15分；02:30 到 04:00 加了 1小時30分；04:00 到 05:45 加了 1小時45分。故下一次應加 2小時0分。05:45 + 2小時 = 07:45)*
  **(答案：07:45)**。
* **過關獎勵：** 獲得虛擬寶物卡片 **【魔法羅勒葉】** + 獲得實體或虛擬提示 **【Wi-Fi 天線碎片 (分為紅色款與藍色款，系統隨機分配)】** + **天線通訊密碼前半段/後半段**。

### ⚡ 關卡 6：小心暴走猩猩 (Tung Tung Tung Sahur)
* **地點建議：** 遊樂器材區。
* **角色對話：** 「{播放名字音檔}咚！咚！咚！我敲了三下鐘，漢堡魔王竟然不理我！氣死我了，我要變身為木頭猩猩了！如果你們不能證明你們有足夠的力氣，我就一棒把你們打飛！」
* **任務內容 (實境體能挑戰 / 拍照任務)：**
  網頁要求：「請全隊一起擺出『大猩猩捶胸頓足』的超兇姿勢，並由同學自拍上傳平台（或由關主認證輸入密碼）。」這是一個歡樂的互動關卡。
* **過關獎勵：** 獲得加工提示紙條：**「收齊麵粉、水、神聖番茄、帕瑪森起司、魔法羅勒葉，前往合成器合作加工」**。

### ⚡ 關卡 7：潮鞋防衛戰 (Tralalero Tralala)
* **地點建議：** 籃球場或球場邊。
* **角色對話：** 「{播放名字音檔} 漢堡魔王踩髒了我的全新限量版 Nike 潮鞋！我要用我的紅色拳擊手套揍扁他！但他設下了『雙色能量防護罩』，單打獨鬥是贏不了的，快去尋找不同顏色的盟友來幫忙！」
* **任務內容 (優化版跨小隊合作題)：**
  *(若小隊尚未通過第5關，網頁會提示：「你需要帶著 Wi-Fi 天線碎片才能啟動此關配對機制，請先去找仙人掌大象 Lirili！」)*
  此關強制啟動實體交流與配對機制，具體執行方式如下：
  1. **尋找盟友：** 網頁會根據你們在第5關拿到的「天線碎片顏色」下達指令。例如拿到「紅色天線」的隊伍，網頁會指示：「請去尋找另一組擁有『藍色天線』的隊伍！」（這解決了不知找誰的問題，只要看顏色，任何一組互補色的隊伍都能配對，也解決了多組獲得天線的問題）。
  2. **密碼組合：** 紅色天線隊伍的手機上會顯示密碼前半段（例如：`SIG`）；藍色天線會有後半段（例如：`MA8`）。兩隊必須將情報結合，並**雙方同時在自己的網頁**輸入完整密碼（`SIGMA8`），才能打破防護罩！
* **過關獎勵：** 啟動天線共鳴，獲得 **【完整 Wi-Fi 分享器權限】** + 漢堡魔王的藏身處座標。

## 四、 結尾與線上合作平台 (Final Mission)

60分鐘的收集時間結束後，所有小隊回到司令台或玄關集合，前往中心的 **「合成協作站」**（實體紙箱改裝，上面有合作 QR Code）。

1. **食材盤點與系統配對：** 小隊掃描合成器 QR Code，系統會檢查是否集齊材料（麵粉、水、番茄、起司、羅勒葉）。
   * 如果「食材不足」，系統將啟動線上互助尋寶機制：系統會自動尋找是否有其他小隊也缺少食材，但剛好跟這組擁有的**食材互補**。
   * **系統提示：** 網頁會跳出通知：「發現盟友！你們可以跟第 X 小隊一起合作湊齊食材！」
2. **終極披薩加工 (線上拖曳介面)：**
   * 當兩組小隊會合並掃描彼此裝置或 QR Code 進入「合作房間」後，手機畫面上會顯示一個「虛擬料理台」。
   * 雙方小朋友可以透過 **拖曳介面 (Drag and Drop)**，將自己手機上的食材拖進中心的「披薩烤盤」中。
   * 當兩邊將所有缺失的食材都拖入放入後，雙方的手機都會同時顯示出最終合成完畢的閃亮亮披薩！
3. **保底/防呆合作機制：** 
   * 如果系統發現「只有這一隊」沒有集齊食材（找不到互補小隊），系統將啟動**防呆救援方案**：隨機指定一隊已經過關的小隊（系統會提示：「快去找第 Y 小隊，他們有多餘的食材願意分享！」）。
   * 當該小組過來協助投入食材後，這組缺少食材的小朋友同樣能看見披薩出爐的畫面，確保**所有小朋友無論如何都能成功通關**並獲得成就感。
