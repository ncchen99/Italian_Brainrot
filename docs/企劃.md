# 🍕 義次元腦洞大開：校園瑪瑪咪呀合成大戰 🤯
（原名：Italian Brain Rot Adventure）

## 國小校園大地遊戲企劃書

---

# 一、活動概念

本活動為一款結合 **校園探索、故事互動與合作解謎** 的大地遊戲。
透過有趣的角色故事與數位互動，小朋友將在校園中尋找隱藏角色，掃描 QR Code 與角色互動、完成任務，並透過不同小隊之間的合作，共同完成一段冒險故事。

本遊戲的核心設計理念為：

**探索（Exploration）**
在校園各處尋找角色與線索。

**互動（Interaction）**
透過手機掃描 QR Code 與角色對話、解謎與完成任務。

**合作（Cooperation）**
不同小隊持有不同資訊，必須互相分享情報才能完成任務。

最終，小朋友們將透過合作解決故事中的危機，讓角色們獲得 Happy Ending，並完成整場冒險。

---

# 二、活動目標

本活動希望透過遊戲設計達到以下目標：

1. **培養團隊合作能力**
   小隊內合作解謎，並與其他隊伍交換資訊。

2. **促進溝通與社交互動**
   鼓勵小朋友之間主動交流線索。

3. **提升探索與觀察能力**
   透過校園尋寶形式增加參與感。

4. **結合數位科技與實體活動**
   使用 QR Code 與互動網頁讓遊戲更具趣味。

5. **創造沉浸式故事體驗**
   讓小朋友透過角色故事參與冒險。

# 四、遊戲流程

整體遊戲將以 **小隊形式進行**，並分為三大階段：**「開關」**、**「冒險過程」**與**「結關」**。

## 階段一：開關（Opening）

### 1. 前情提要與 AI 影片開場
* 讓所有小朋友集合於視聽教室或禮堂。
* 播放一段由 **AI 生成的開場影片**，向小朋友介紹 7 位義次元角色的背景故事，並交代他們目前遇到了什麼危機（例如：魔法材料散落各地、記憶被偷走等）。
* 指派任務目標，點燃小朋友們拯救角色的使命感。

### 2. 分組與任務派發
小朋友會被分成若干小隊，例如：
* 每隊 3–5 人
* 每隊會收到一份「任務包」（包含地圖、第一張材料卡或線索）
* 每一隊會被指派一個特定的「主要角色任務」起點。

---

## 階段二：冒險過程（Adventure） - 收集食材階段 (約 60 分鐘)

### 1. 自由探索與收集
校園中會放置 **七個角色點位** 與若干隱藏點：
* 每個點位包含：角色圖片、專屬 QR Code。
* 小朋友有 60 分鐘的時間，根據手中的地圖與線索，滿校園跑動去挑戰所有關卡，盡可能收集到做披薩的「食材卡片」與線索。

### 2. QR Code 互動與解謎
掃描 QR Code 後，會開啟網頁互動頁面。

可能包含：

**角色對話**

例如：

> 我需要找到三個符號才能恢復力量。

**小遊戲或解謎**

例如：

* 選擇題
* 排序題
* 找線索
* 拼圖
* 簡單邏輯題

**🧩 闖關任務與解謎成分**
為了讓遊戲更具挑戰性與延長遊玩時間，闖關任務將具備**複雜度與解謎成分**：
* **跨關卡解謎**：在 A 地點獲得謎題的「上半部」，必須在 B 地點找到「下半部」才能推導出密碼。
* **邏輯推理題**：提供一份「角色之間的證詞」，小朋友必須像偵探一樣推敲出誰在說謊。
* **實境解謎**：網頁提示「數數看這個川堂有幾根柱子並輸入數字」，小朋友必須在實體校園中尋找答案。

**⏳ 角色互動冷卻機制 (Cooldown System)**
為了拉長遊戲進程並避免小隊過度集中在同一個關卡，系統將加入**冷卻時間（Cooldown）**：
* 當小隊與某個角色互動後（特別是若挑戰失敗），該角色點位對該小隊會進入 **X 分鐘的冷卻狀態**。
* 冷卻期間掃描 QR Code 會顯示角色專屬的幽默嘲諷畫面（例如角色喝著咖啡說：「急什麼？大腦還沒降溫呢！先去別的地方轉轉吧！」）。
* 此機制鼓勵小朋友在冷卻期間去探索其他點位或與其他隊伍交換情報。

完成任務後，小隊會獲得：

* 新線索
* 虛擬寶物（材料卡牌）
* 情報

---

## 階段三：集合與合成 (Assembly & Final Mission)

當 60 分鐘的探索時間結束後，所有小隊將回到最初的集合地點，前往「合成協作站」。

### 1. 播放 AI 結尾影片
* 播放由 **AI 生成的結尾影片**，展示所有角色因為小朋友們的幫助而恢復原狀，甚至吃到了最美味的「瑪格麗特披薩」！
* 影片中可以加入對小朋友合作精神的稱讚，為故事畫下完美的 Happy Ending。

### 2. 頒獎與大合照
* 頒發紀念證書（見第九章：活動獎勵）。
* 全體小朋友與螢幕上的義次元角色大合照。

# 五、虛擬寶物與合成系統（Minecraft 風格）

本遊戲將導入收集與合成要素，讓尋寶過程更有成就感：

### 1. 虛擬寶物與材料呈現
* 收集到的線索或道具不再只是純文字，而是以**類似卡牌的虛擬寶物**呈現在手機畫面上（例如：🌟閃亮的起司卡、🍅神聖番茄卡）。
* 每張卡牌都有專屬的幽默說明與插圖，增加收集樂趣與成就感。

### 2. 收集與合成系統
* 遊戲借鑑《當個創世神》（Minecraft）的合成概念，小朋友需要先收集五種基礎材料卡（麵粉、水、番茄、起司、羅勒葉）。
* 當五種材料都齊全後，即可在合成器頁面進行合作加工，不需先額外合成「生麵團」。

### 3. 合成指示的隱藏與獲取
合成配方（例如 材料 A + 材料 B = 材料 C）不會直接白白送給玩家，必須透過以下方式取得：
* **角色對話彩蛋**：小朋友必須找到並與特定角色對話，角色才會不經意透漏：「聽說把起司和番茄放在一起會發生神奇的事...」
* **校園隱藏 QR Code**：將合成公式做成隱藏版的 QR Code「配方碎片」，藏在校園的佈告欄角落、樹下等隱密處供小隊尋找。

### 4. 實體「合成器」互動
* 校園內將設置一至數個具體的**實體「合成器」站點**（可以用紙箱裝飾成神秘機器的模樣，充滿儀式感）。
* 小朋友必須親自走到「合成器」現場，掃描該機器專屬的 QR Code，才能開啟手機上的「合成介面」進行材料合成。
* 這樣的實體互動可以大幅增加遊戲儀式感，並擴展小朋友的跑動範圍與時間。

# 六、合作機制與結盟系統

本遊戲鼓勵 **合作式遊戲體驗**，跳脫傳統的「零和競爭」，最關鍵的機制發揮在遊戲最後的「線上合作平台」階段：

1. **系統自動配對互助：** 
   回到集合點掃描「合成器」後，若系統發現**食材不足**，會主動尋找「也缺少食材且品項互補」的他組小隊，並提示：「您可以跟 X 組的小隊一起合作」。
2. **互動式拖曳介面設計：**
   當雙方會合後，只要進入合作頁面，各自分別拖入擁有的食材卡片。當所有食材都放入烤爐時，兩邊的手機都能同時見證披薩完成的瞬間！
3. **保底通關機制：**
   如果系統匹配不到互補小隊（例如全場只剩下一隊沒收集完），系統會彈性提示：「請找任一已通關的小隊請求支援！」只要該隊伍過來點擊投入食材，這組小朋友也能順利獲得披薩，確保每個人都能快樂破關。
4. **全體大合作：**
   （備案）若全場都缺少某樣關鍵食材，關主也將啟動「全體大合作機制」，讓全場小朋友一起挑戰大任務贏回最後一塊材料。

這樣結合線上 UI 互動的設計能確實引導小朋友進行交流，就算闖關失敗，也能體會到「團結力量大」的美好結局。

# 七、故事結構

整體故事會由 **七個角色任務串聯而成**。

每個角色：

* 遇到某個困難
* 需要特定資訊或線索
* 與其他角色的任務相關

在遊戲後期，小朋友們會逐漸發現：

> 七個角色的問題其實彼此相關。

最終：

所有小隊需要整合各自找到的線索，
共同完成最後一個任務。

---

# 八、最終任務（Final Mission）

在完成所有角色關卡後，會進入 **最終關卡**。

這個關卡需要：

* 所有小隊的資訊
* 收集到的符號或線索
* 合作解開最後的謎題

當最後謎題完成後：

故事中的危機被解決，
所有角色恢復正常。

小朋友成功完成整場冒險。
