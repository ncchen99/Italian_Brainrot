# 義次元腦洞大開（Italian Brainrot）

一個給校園活動使用的 **手機網頁闖關遊戲**。  
玩家（小隊）要在限時內掃描校園 QR Code，完成 7 個角色關卡，收集披薩食材，最後進入合成站合作完成終極任務。

## 這個遊戲在做什麼

- 以「瑪瑪咪呀學園」為舞台的實境解謎闖關
- 每個關卡對應一個角色與一段互動任務（排序、密碼、拍照、協作等）
- 系統記錄每隊進度、掃碼存取權限與剩餘時間
- 最終關卡會檢查食材是否齊全，若不足可透過跨隊合作補齊

## 劇情設定（簡版）

漢堡魔王入侵校園，藏起免費 Wi-Fi，並把「終極瑪格麗特披薩」所需材料打散到各角色關卡。  
小隊要在 60 分鐘內完成任務，蒐集：

- 麵粉
- 水
- 番茄
- 起司
- 羅勒葉

最後前往合成協作站完成加工，恢復校園訊號並達成結局。

## 使用者要完成什麼

1. 進入系統後建立小隊身份
2. 在校園掃描關卡 QR Code 進入對應關卡
3. 完成各關任務，累積食材與進度
4. 時間結束前到合成站完成最終協作
5. 觸發結局並完成整體任務

## 關卡流程概覽

- Level 1：忍者的修煉（記憶/順序）
- Level 2：舞鞋在哪裡（實境尋寶 + 密碼）
- Level 3：吵鬧的青蛙（限時點擊）
- Level 4：水球空投警報（邏輯推理）
- Level 5：沙漠與斷網的絕望（時間數列）
- Level 6：小心暴走猩猩（拍照/認證）
- Level 7：潮鞋防衛戰（跨隊密碼合作）
- Synthesis：食材盤點與合作合成

## 專案結構

- `app/`：前端 React + Vite 專案（主要程式碼）

- `assets/`：遊戲素材與 QR 圖檔
- `scripts/`：素材處理工具
